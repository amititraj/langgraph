# langgraph

RAG Chat and Blog Writer are projects created using LangGraph.

# RAG Chat

RAG Chat is a small retrieval-augmented generation (RAG) application that demonstrates how to build a question-answering/chat system backed by 
a local vector store and document processing pipeline.

# Blog Writer

Blog Writer is an AI-powered application for generating high-quality, informative, and engaging blog content based on user-provided topics. 
It uses LangChain, OpenAI, and web search tools to create well-structured blog posts in Markdown format.
